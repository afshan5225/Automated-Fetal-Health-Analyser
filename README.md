# Automated Fetal health analyser

## Overview

This project automates the analysis of pregnancy scan reports using Natural Language Processing (NLP) and Optical Character Recognition (OCR). The aim is to extract and evaluate key medical entities from scanned reports to determine fetal health based on predefined reference ranges.

## Features

- Train a Named Entity Recognition (NER) model using spaCy.
- Extract text from scanned images using EasyOCR.
- Compare extracted data against reference ranges to assess fetal health.

- open the folder , in cmd type "python run app.py"
- upload trimester reports from the input data.
